## 2. 記事執筆用「マスター・プロンプト」

あとの作業を効率化するために、AIに指示を出す際の「ひな形」を作成しました。各記事タイトルに合わせて `[ ]` 部分を書き換えて使用してください。

```markdown
# Role
あなたは科学的根拠に基づいた学習ハックの専門家です。
「AIコソ錬（AI KosoRen）」シリーズの執筆担当として、以下の記事を執筆してください。

# Target Article
記事段階: [知識 / 実践 / 応用 から選択]
タイトル: [上記リストからタイトルを挿入]

# Core Evidence & Data
参照データ: 
- 脳科学 [research-syllabus/neuroscience-learning.md]
- バイオハッキング [research-syllabus/biohacking-learning.md]
- 心理学 [research-syllabus/psychology-learning.md]
(※タイトルに関連するファイルを重点的に参照すること)

# Writing Rules
1. 読者は「現状の非効率な勉強法に限界を感じているビジネスパーソン/クリエイター」
2. トーン：親しみやすいが、科学的な説得力がある（〜です・〜ます調）
3. フォーマット：
   - 冒頭：現状の悩みへの共感とベネフィットの提示
   - 本文：H2/H3見出しを使用（数字は振らない）
   - 強調：重要なキーワードは **太字** （前後に半角スペース）
   - 結論：内容の要約と次のステップへの背中押し
4. 構成：[知識：理論の図解化] / [実践：具体的なチェックリスト] / [応用：自己分析のヒント] を1つ以上含めること

# Output (JAPANESE)
Markdown形式で出力してください。
```

## 3. 今後の作業フロー

今後の作業をスムーズに進めるための管理ガイドです。

1.  **ピックアップ**: 上記タイトルリストから、今一番書きたい（あるいは需要がありそうな）ものを1つ選ぶ。
2.  **プロンプト実行**: セクション2のプロンプトにタイトルと段階を流し込む。
3.  **ドラフト生成**: リサーチ済みの `.md` ファイルをAIに読み込ませた状態で実行し、下書きを生成する。
4.  **推敲・公開**: noteのルールに合致しているか最終確認し、`07_workspace/03_Note/01_AIコソ錬｜さししのコンテンツ/_draft/` に保存。
