# 脳科学観点のリサーチ：学習効率の最適化

脳科学に基づいた最適な学習方法として、以下のエビデンスが重要視されています。

## 1. 分散学習（Spaced Learning）
短期間の集中学習（詰め込み）よりも、適切な間隔を空けて復習する方が長期記憶に定着します。

- **エビデンス:** エビングハウスの忘却曲線に基づき、忘れかけた頃に思い出す「望ましい困難」が脳を刺激し、記憶を強化します。
- **具体策:** 翌日、3日後、1週間後と間隔を広げて復習をスケジュールする。

## 2. インターリービング（Interleaving）
一つの科目を長時間続けるのではなく、関連する複数の科目や問題を混ぜて学習する手法です。

- **エビデンス:** 脳に変化を与えることで集中力を維持し、知識の「識別能力」や「応用力」を向上させます。
- **具体策:** 20分おきに異なるジャンルの問題を解く。

## 3. アクティブ・リコール（想起練習）
受け身のインプットよりも、自力で思い出すアウトプットが記憶定着に直結します。

- **エビデンス:** 脳は「思い出す努力」を要する情報を重要と判断し、長期記憶に刻み込みます。
- **具体策:** テスト形式、フラッシュカード、学んだことを白紙に書き出す。

## 4. 睡眠の役割
学習後の睡眠は、記憶を固定化（コンソリデーション）するために不可欠なプロセスです。
