# SNS戦略：Syllabus Hack 認知拡大・インデックス促進プラン (詳細版)

## 🎯 目的
- **インデックス促進:** SNS（特にリンクをクリックしやすいThreads/X）からの流入を発生させ、Googleボットの巡回を促す。
- **認知拡大:** 開設間もないドメインの弱さをSNSの拡散力でカバーし、初期ユーザーを獲得する。
- **運用継続性:** ストック記事（100+）を有効活用し、低コストで運用する。

## 🚀 運用体制 & チャネル選定

### 1. 投稿主体
- **メインアカウント:** さしし (@sasisi344)
- **理由:** E-E-A-T（信頼性）の確保、および既存フォロワーへの初動リーチを最大化するため。

### 2. チャネル別役割
| チャネル | 役割 | 予約/手動 | 備考 |
| :--- | :--- | :--- | :--- |
| **Threads (注力)** | **予約投稿ハブ** | 予約 (12:00/18:00) | 記事リンクが踏まれやすく、予約機能が使いやすいため。 |
| **X (Twitter)** | **拡散・リアルタイム** | 手動 / 簡易投稿 | API制限のため、Threadsの内容をコピペ、または「今読んでほしい」時に投稿。 |
| **Instagram** | **ビジュアル・カタログ** | 予約 (12:00) | 記事カバー画像＋要約スライドで「保存」を狙う。FB Business経由。 |

## 📅 固定スケジュール (Time Batching)

ルーチンとして投稿時間を固定し、学習者のアクティブ時間に合わせます。

- **12:00 (昼休み):** 
  - **内容:** 「短時間で学べるTips」系 (Methodカテゴリ)。
  - **ターゲット:** 移動中や昼休みの学習者。
- **18:00 (帰宅/夜活):**
  - **内容:** 「キャリア・転職の現実」系 (Careerカテゴリ)。
  - **ターゲット:** 仕事終わり・今後のキャリアを考える社会人。
- **週末 (土日):**
  - **内容:** 「クイズ配信」または「今週の注目記事まとめ」。

## 📝 ポスト構成テンプレート

### 1. Threads / X (ハブ用) アルゴリズム最適化
- **本体ポスト:** **URLを載せない**（外部サイトへのリンクがあるとリーチが激減するため）。
- **画像:** 記事独自のカバー画像 (`cover.jpg`) を**手動で添付**（OGPに頼らない）。
- **リプライ欄:** 本文に関連する補足や「詳細はプロフリンク or こちら」として**URLをリプライの1件目に貼る**。

#### 構成案:
- **1行目 (フック):** キャッチーな問いかけ
- **2行目 (中身):** 3つの弾丸ポイントで要約
- **3行目 (誘導):** 「詳細はリプライ欄へ👇」
- **添付:** カバー画像
