# Bold 記法を strong タグに変換

対象ファイル: $ARGUMENTS

`**bold**` 形式のMarkdown bold を `<strong>bold</strong>` HTML タグに一括変換してください。

## 手順

### Step 1: 対象ファイルの確認

- 引数が特定ファイルパスの場合: そのファイルのみ処理
- 引数が `all` またはディレクトリパスの場合: 配下の `*.md` / `*.mdx` を Glob で取得
- 引数が空の場合: `.workspace/draft/` 内の全ファイルを処理

### Step 2: 変換前プレビュー

対象ファイルを Read して、変換対象の `**bold**` 箇所を列挙する（Grep パターン: `\*\*[^*]+\*\*`）。

変換件数と変換内容をユーザーに提示して確認を取ること。

### Step 3: 変換実行（確認後）

ユーザーが確認した後、以下のルールで変換する:

**変換パターン:**

- `**テキスト**` → `<strong>テキスト</strong>`

**注意事項:**

- コードブロック（` ``` ` で囲まれた部分）の中は変換しない
- インラインコード（`` `コード` ``）の中は変換しない
- frontmatter（`---` 〜 `---` の間）は変換しない

### Step 4: 変換後の検証

変換後のファイルを Read して残存する `**` がないことを確認する。

### Step 5: 完了報告

- 変換したファイル数
- 変換した `<strong>` タグの総数
- 変換できなかった（スキップした）箇所があれば明示
