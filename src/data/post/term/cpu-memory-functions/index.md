## 3行まとめ
*   <strong>CPU (中央処理装置)</strong>: コンピュータの「頭脳」。計算や制御を担当する。
*   <strong>メモリ (主記憶装置)</strong>: 処理中のデータを一時的に置いておくための「作業机（ワークエリア）」。
*   <strong>関係性</strong>: メモリ（机）が広ければ広いほど、CPU（人）は効率よく作業を行えます。

## 試験での出題ポイント
試験では、CPUの「性能」とメモリの「種類」が問われます。

1.  <strong>CPUの性能</strong>: <strong>クロック周波数</strong>（処理スピード）や、 <strong>マルチコア</strong>（同時に作業できる人数）などで決まります。
2.  <strong>メモリ (RAM)</strong>: 電源を切るとデータが消える <strong>揮発性</strong> を持っています。
3.  <strong>キャッシュメモリ</strong>: CPUとメモリの速度差を埋めるために、CPU内に設けられた高速な一時保管場所です。
4.  <strong>レジスタ</strong>: CPUの内部にある、極めて高速な一時記憶場所のことです。

## まとめ：効率を高める「脳」と「机」のバランス
CPU（脳）がいくら速くても、メモリ（机）が狭ければ作業は進みません。逆にメモリが広くてもCPUが遅ければ指示が追いつきません。

試験では、 <strong>演算・制御</strong> を行うのがCPU、 <strong>一時記憶</strong> を行うのがメモリ、という基本の役割分担を確実に覚えましょう。
