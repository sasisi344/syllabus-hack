## なぜ「ストラテジ系」を集中的に学ぶべきなのか

ITパスポート試験の全100問のうち、ストラテジ系は <strong>約35問</strong> を占めます。つまり、全体の <strong>配点の35%</strong> がこの分野に集中しています。

しかも、ストラテジ系は範囲が広い。企業活動の基礎から経営戦略、法務、最新のAI倫理まで、ビジネスの「常識」が問われます。

ここで重要なのは、ストラテジ系の知識は <strong>試験のためだけのものではない</strong> ということです。

### 実務で即戦力になる知識

- <strong>経営戦略マネジメント</strong> ：PPM、SWOT分析、マーケティング手法など、ビジネスの共通言語を習得できる
- <strong>法務</strong> ：個人情報保護法、知的財産権、労働関連法規など、社会人として知っておくべき法的知識
- <strong>企業活動</strong> ：財務諸表の読み方、データ分析の基礎など、どの職種でも求められるリテラシー
- <strong>ビジネスインダストリ</strong> ：AI、IoT、DXなど、最新テクノロジーのビジネス応用

つまり <strong>ストラテジ系を制する者は、試験もキャリアも制する</strong> と言っても過言ではありません。

### 向いている学習デバイス

ストラテジ系の長文設問は、PCまたはタブレット横置きで読むと負荷が下がりやすい。スキマ演習はスマホでも可能だが、メニュー設計は [ITパスポートはスマホ学習で合格できる？スキマ時間で進める最短ルート](/method/smartphone-study-guide) を参照。模試の通しは [PC学習が強い理由。長時間学習で合格率を上げる環境の作り方](/method/pc-study-setup-guide) に寄せると本番感覚に近づきやすい。全体の整理は [スマホ・タブレット・PCをどう使い分ける？IT資格学習の最適デバイス戦略](/method/multi-device-cert-study) にまとめている。

### このドリルの特徴

このドリルには、ストラテジ系の問題を <strong>101問</strong> 収録しています。

| 中分類               | 問題数 |
| -------------------- | ------ |
| 企業活動             | 33問   |
| ビジネスインダストリ | 26問   |
| 経営戦略マネジメント | 20問   |
| 法務                 | 10問   |
| システム戦略         | 8問    |
| 技術戦略マネジメント | 3問    |
| システム企画         | 1問    |

### 苦手問題の優先出題アルゴリズム

普通のドリルアプリとの最大の違いは、<strong>苦手な問題ほど出やすくなる</strong> こと。

- 初回は <strong>未回答の問題が優先的に</strong> 選ばれ、まんべんなく出題される
- 間違えた問題は <strong>次回以降のセッションで出題確率が5〜8倍</strong> に上がる
- 正解した問題は出題確率が下がるが、ゼロにはならない（忘却防止）

つまり、繰り返すほどあなたの弱点だけを集中攻撃する <strong>オーダーメイドのドリル</strong> になります。

### 使い方

1. まず <strong>30問、40問、50問</strong> のいずれかを選んでスタート
2. 解答後すぐに正誤と <strong>AI解説</strong> が表示される
3. セッション終了後に <strong>中分類ごとの正答率</strong> を確認
4. 「もう一度挑戦」ボタンで、苦手が優先された新しいセットに挑戦
