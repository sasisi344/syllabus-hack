## 専門用語の暗記を「作業」から「ハック」へ

情報処理安全確保支援士（SC）やITパスポート、基本情報。情報処理技術者試験の合格を阻む最大の壁、それは <strong>膨大な専門用語の暗記</strong> です。
「問題演習は楽しいけれど、用語を覚えるのが苦痛…」そんな声に応えるために開発したのが、この <strong>暗記ハック（Flashcards）</strong> です。

### このアプリの特長

1.  <strong>SC試験・最新シラバス対応</strong> : SCの午前II・午後対策に特化した重要用語を収録。
2.  <strong>2-Way学習モード（SC）</strong> : 「用語から説明」だけでなく「説明から用語」を思い出す逆引き学習で、記述力も同時に鍛えます。
3.  <strong>インプット特化のUX</strong> : 4択問題よりも高速に、テンポよく用語をチェックできます。
4.  <strong>隙間時間の活用</strong> : PCでもスマホでも、片手でサクサク学習が進みます。

### 推奨される学習フロー

1.  <strong>インプット</strong> : このフラッシュカードで、知らない用語を「脳にインデックス」します。
2.  <strong>仕分け</strong> : 覚えたカードは「✓」、不安なものは「✕」で仕分け、効率的に反復します。
3.  <strong>アウトプット</strong> : 用語を覚えたら、[生成AI・AI倫理クイズ](/app/genai-ethics-quiz)などのドリルで実戦力を試しましょう。
