# Syllabus Parser

This tool helps structure the raw text from IPA Syllabus PDF into the JSON format required for `syllabusRef`.

1. Select the <strong>Target Exam</strong> (IP, SG, FE).
2. Select the <strong>Middle Category</strong> you are working on.
3. Paste the <strong>Keywords</strong> section from the PDF (e.g., the text column under "用語例" or "技術要素").
4. Click <strong>Extract</strong>.
5. Repeat for other categories.
6. Click <strong>Copy Full JSON</strong> to get the `syllabus-{examId}.json` content.
