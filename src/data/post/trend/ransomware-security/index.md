## 試験での出題ポイント
試験では、「ランサムウェアに感染した際の症状」や「感染を防ぐ・被害を最小限にするための具体的行動」が問われます。
1.  <strong>侵入経路</strong>: 標的型メールの添付ファイル、VPN機器の脆弱性など。
2.  <strong>被害最小化</strong>: 感染が疑われたら即座にネットワークから物理的に切り離す。
3.  <strong>EDR / XDR</strong>: 不審な動きをいち早く検知して、自動で遮断する最新の仕組み。
4.  <strong>二重脅迫</strong>: データを暗号化するだけでなく、「公開されたくなければ払え」と脅す手法。
