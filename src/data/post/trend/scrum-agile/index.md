## 試験での出題ポイント
試験では、「誰が・何に責任を持つか」という役割の違いを問う問題が非常に出やすいです。
1.  <strong>PO</strong>: 何を作るか、優先順位を決める（ビジネス価値の責任者）。
2.  <strong>SM</strong>: チームが円滑に動けるようサポート、障害を取り除く（ファシリテーター）。
3.  <strong>開発チーム</strong>: 実際に動くソフトウェアを作る（自己組織化されたチーム）。
4.  <strong>レトロスペクティブ</strong>: スプリントの最後に、自分たちの働き方を振り返る。
