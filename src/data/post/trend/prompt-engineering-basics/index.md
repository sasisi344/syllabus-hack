## 試験での出題ポイント
試験では「どのようにプロンプトを構成すれば、より精度の高い回答が得られるか」という具体例を問う問題が出やすいです。
1.  <strong>役割（Role）</strong>: 「あなたはプロのエンジニアです」と役割を与える。
2.  <strong>文脈（Context）</strong>: 「新入社員向けの資料作成のために」と背景を伝える。
3.  <strong>Few-shot</strong>: 数個の「具体例」を提示して、形式を学習させる技術。
4.  <strong>Chain-of-Thought (CoT)</strong>: 「ステップバイステップで考えて」と指示し、思考過程を出力させる手法。

## 【AIハック】生成AIで最速暗記
AIを使って「プロンプトエンジニアリング」自体を学ぶと一石二鳥です。

> 「ITパスポート試験の『Few-shotプロンプティング』について、小学生でも分かるように例え話を使って説明してください。回答は150文字以内で、最後に応援メッセージを付けてください。」
