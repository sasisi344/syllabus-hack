<h3>3行まとめ</h3>
<ul>
<li><strong>XAI（Explainable AI）</strong>は、AIが「なぜこの結論に至ったか」という根拠を人間が理解できるように説明する技術の総称。</li>
<li>ITパスポート試験では、最新のAIガバナンス（社会的な信頼性・安全性）の文脈で頻出。</li>
<li>中身が不明な「<strong>ブラックボックス</strong>」状態を解消し、社会が安心してAIを利用できるようにする仕組み。</li>
</ul>
</div>

<h2>シラバス上の位置付け</h2>
<ul>
<li><strong>テクノロジ系</strong> / 基礎理論 / AI（人工知能）理論</li>
<li><strong>ストラテジ系</strong> / 企業活動 / AIガバナンス（信頼性）</li>
</ul>

<h2>試験での出題ポイント</h2>
<p>試験では、「AIが意思決定に深く関わる場面（自動運転、医療診断、銀行の融資審査など）」におけるXAIの役割が問われます。</p>
<ol>
<li><strong>信頼性の向上</strong>: AIの答えをうのみにせず、人間が「妥当かどうか」を最終判断するための材料を提供。</li>
<li><strong>責任の所在</strong>: 不具合や事故が起きた際、「なぜその判断をしたか」を遡って調査（監査）し、改善するために必要。</li>
<li><strong>公平性の確保</strong>: 学習データに含まれるバイアス（偏見）により、特定の人を不当に差別していないかをチェックする。</li>
</ol>

<h2>【AIハック】生成AIで最速暗記</h2>
<p>AIに、自分自身の「判断の癖」を説明させる練習をしてみましょう。</p>

<div class="prompt-box">
<h3>プロンプト例：</h3>
<p>「ある銀行で融資審査AIが『年収○万円以上という理由でOKにしました』と説明することを『XAI』と呼びます。この説明が一切ない場合に、顧客や銀行側が直面するリスクを1つ、分かりやすく教えてください。」</p>
</div>

<p><strong>合格へのヒント：</strong><br />
「精度が高ければ中身はどうでもいい」という時代から、「正しく説明できなければ使えない」という時代に変わった、と捉えるのがXAI理解のポイントです。</p>
