<h3>3行まとめ</h3>
<ul>
<li>「<strong>適格請求書（インボイス）</strong>」という、正しい消費税率や登録番号が記載された請求書を保存・発行する制度。</li>
<li>ITパスポート試験では、業務プロセス（購買から会計まで）の変化や、ITシステムの対応要件として問われる。</li>
<li>企業の「ERP（統合基幹業務システム）」などが、制度変更にどう適応すべきかが合否の分かれ目。</li>
</ul>
</div>

<h2>試験での出題ポイント</h2>
<p>試験では、「インボイス制度導入によって、システムにどのような具体的機能が必要になるか」が問われます。</p>
<ol>
<li><strong>請求書の形式変更</strong>: 発行者の登録番号、適用税率（8%・10%）、税率ごとの消費税額を自動計算して表示する機能。</li>
<li><strong>電子帳簿保存法との連携</strong>: インボイスの電子データを、法令で定められた検索要件や改ざん防止要件を満たした状態で保存する仕組み。</li>
<li><strong>DXの加速</strong>: 手作業の請求処理をなくし、電子データ（<strong>EDI</strong>）での自動やり取りを推進するきっかけとしての役割。</li>
</ol>
