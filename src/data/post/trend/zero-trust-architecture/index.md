## 試験での出題ポイント
試験では、「なぜゼロトラストが今の時代に必要なのか」という理由（クラウド、テレワーク等）と、検証される項目の事例が問われます。
1.  <strong>どこからでも</strong>: 自宅、カフェ、会社。アクセス元がどこでも、同じ強度の検証を行う。
2.  <strong>すべてのデバイス</strong>: 接続してくるPCやスマホが「安全か（ウイルス感染していないか）」を都度チェックする。
3.  <strong>最小権限</strong>: ユーザーに必要なサービスやデータだけを、必要な時だけ許可する。
4.  <strong>境界防御の崩壊</strong>: VPN等の「入り口さえ守ればOK」という手法では防げない攻撃への対策。
